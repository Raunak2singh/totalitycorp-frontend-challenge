# E-commerce Website with Next.js

This is an e-commerce website built using Next.js and a variety of other technologies.

## Technologies Used

- **Next.js**
- **@mui/material**
- **bcryptjs**
- **js-cookie**
- **jsonwebtoken**
- **mongoose**
- **next-connect**
- **notistack**
- **react-hook-form**
- **React and ReactDOM**
